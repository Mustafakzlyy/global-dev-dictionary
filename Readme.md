# 📖 Global Dev Dictionary

A beginner-friendly, open-source web application designed to help developers make their very first Pull Request! This project serves as a multilingual dictionary for common programming and software terms.

---

## 🚀 How to Contribute

### 1. Add or Update Terms
1. Go to the `locales/` folder.
2. Open your native language file (e.g., `tr.json` for Turkish, `en.json` for English). If your language doesn't exist, feel free to create it!
3. Add a software term with its definition using the proper JSON format.

### 2. Improve the Website (Do Whatever You Want!)
If you want to hack the frontend rather than just adding words, you have total freedom! You can:
- Improve the UI/UX with better CSS or animations.
- Add a dark mode toggle.
- Implement a live search bar in `app.js` to filter terms easily.
